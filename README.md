

# project structure

```shell
$project
├── README.md
├── back-end
│   ├── dist
│   ├── nest-cli.json
│   ├── node_modules
│   ├── package-lock.json
│   ├── package.json
│   ├── src
│   ├── test
│   ├── tsconfig.build.json
│   └── tsconfig.json
└── front-end
    ├── node_modules
    ├── package-lock.json
    ├── package.json
    ├── public
    └── src
```

# How to start

```shell
cd $project/back-end
npm install
npm run start

cd $project/front-end
npm install
npm run start
```
